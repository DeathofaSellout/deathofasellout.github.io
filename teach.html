Become A Full Stack Web Developer

There are a lot of careers in the world of tech, but this is a career where you infuse tech and creativity. People can see what you've created and you can help change their lives for the better.

Web Development

There are a lot of different roles in the tech world. For me, the most creative is Full Stack Web Development. You can create your own web apps. You are only limited by your own genius. Build apps to help people, build apps to help yourself, and/or build apps to make money. Whatever your decision may be, getting started can be rough. That is why I am passing on my knowledge. You can use this knowledge to completely teach yourself or you can use this path to establish a working knowledge of javascript to enter into a bootcamp.

Different Tech Stacks

There are a lot of different Tech Stacks. A Tech Stack is a full stack of web technologies you use to build a web app, divided into Front End and Back End. Front End is what you see, Back End is where are all of the information is stored. A web app is a website like Facebook or Google. It's not the exact same thing as building an IOS app or Android app. Each of those are their own careers in tech. The two tech stacks that I am experienced with are Ruby on Rails and the MERN/MEAN tech stacks. Yes there are others, but focus on one, then you can learn any other. For now, we are going to focus on the MEAN/MERN stack. MERN stands for MongoDB, Express, React, and Node js. MEAN stands for Mongo DB, Express, Angular, and Node js. These are Javascript based Full Stacks. You will be using the MVC architechtural paradigm: Model-View-Controller.

Learn How To Work From A Command Line Interface/Linux or Unix based

Linux/CLI

You need to learn how to navigate your computer from a terminal. Using a Mac or Linux based OS provides you with a terminal, typically a BASH terminal. Creating files, directories, running a node server, installing everything you need for an app; all of this is done from a command line. Learn it, love it, use it.

Learn GIT

GIT

Learn GIT, by extension learn how to use github. This is where you store your projects. This is where companies store their projects. Git is a version control system for tracking changes in computer files, for coordinating work on those files with multiple people or for yourself.

github.com

This is the most prominent git repository. This is where you store your projects. This is where companies store their projects. This is where you can get ruby gems to use, where you can get premade MERN stack or MEAN stack templates. A place for companies look at your code to see what you are doing.

Using Google Chrome Dev Tools

Chrome Dev Tools

This is a link to learn how to use Chrome Dev Tools. You can open it on any website displayed on your browser (Google Chrome Only). It allows you to have a console for testing code, to see errors, to examine your HTML and CSS, to see what other websites code looks like. You will use it often.

Start with Javascript

This is the absolutely most important step in your whole journey, one you will work on your entire career. Start with a object oriented programming language before you learn HTML and CSS. Why? Learning an object oriented programming language is a lot harder to learn than HTML and CSS. The sooner you get started the better. Also, once you learn a OOP language, it's a lot easier to learn other OOP languages because a lot of them are the exact same concepts, just done differently with different syntax. Coding is hard, it's not easy. Spend one to two hours every day learning. You may hit a wall when you are first getting started, but know this: it will all make sense. It will also become a lot of fun. It's all about problem solving and becoming a better critical thinker. Well, where to start?

jshint.com

This is a site that helps check your code to see if you are missing any of the basics. Such as commas, semicolons, and parentheses. Seems trivial, but it is crucial when you are first getting started. Always check your code if it's not running in your console/text editor.

Learn Higher Order Functions

Intro to HOFs

A HOF is a function that accepts another function as an argument. Fancy For Loops that do wondeful things for you. Learn them and love them. Also, the guy in the above article has a wonderful youtube channel that is great for learning about JS in all of it's glory.

Learn Recursion

Intro to Recursion

Recursion is a rabbit hole you want to go down. It's a function that calls itself until it solves a problem by solving parts of a problem, one at a time. You need a base case and then a recursive call. Read the above article to start down the path of recursion.

HTML & CSS

Hyper Text Markup Language (HTML) is like the bones, the physical framework for your website. It's the text you are reading right now and how it is arranged. It's how your whole website is arranged. Cascading Style Sheets (CSS) is the styling of your website. Changing how your text looks, how your images are presented, and the colors you see. CSS can also do more.
