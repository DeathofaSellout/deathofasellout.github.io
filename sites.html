Web Apps and Websites I Have Created

LearnToCode

This is a social networking app that teaches a person how to become a full stack web developer. Create an account. Inside you have three pages that teach what tools you need to get started, what to learn in javascript, and where to go to learn how to build full stack web apps. There is a Forum where a user can ask questions and get help. There is a place to create Tasks to help keep track of what a user has learned and what they are working on. Finally, they can contact for me for help.

This app consists of Ruby on Rails, Ruby, Materialize, jQuery, postgres, Active Record, HTML, and CSS. Deployed on Heroku. I am the sole creator and developer of this site.

ComeShred

This is a social networking web app for extreme sports. There is a map where you can find new spots or add spots to skate, surf, snowboard, parkour, scooter, bmx, or whatever extreme sports you do. You can add friends and message each other. There is also a page for posting about your sports, events, and/or whatever else is related to extreme sports.

This app consists of Ruby on Rails, Ruby, Materialize, jQuery, postgres, HTML, CSS, Active Record, and a lot of different gems. Gems such as bcrypt, geocoder, gmaps4rails, underscore, devise, and mailboxer. Deployed on Heroku and connected to Google Domains. I am the sole creator and developer of this site.

Vagabond

This is a web app for a traveling agency. Different cities have their own pages you can post about. You can create your own account and see all of your own posts. This app was created by Jonathan Mules, Huan Ming Liao, and I. This app consists of Ruby on Rails, Ruby, Bootstrap, jQuery, HTML, CSS, Javascript, and different gems. Gems such as bcrypt and FFaker. Deployed on Heroku.

BeerApp

This is a web app for beer enthusiasts. You can find new beers or add new beers to the site's list. You can also rate beers for others to see. This app was created by Bryan Reese, Jonathan Mules, and I. Technologies that were used are Ruby on Rails, Ruby, jQuery, Ransack gem (for searching), Ratyrate gem (for star rating), and Materialize.

MicDrop

This is a web app for creating playlists. Created by my friend Bryan Albini and I, it's a MEN stack app. Simple, but one our first apps. We used Mongo DB, Mongoose, Bootstrap, Javascript, EJS, and Express. It works as a single page app, even though there is no front end framework! The other creator and I are working on turning it into a Facebook app. Deployed on Heroku.

Colorist Website

This is a website I created for a friend using godaddy and their website builder. I stylized and changed around their components. Simple and easy, was my first experience publishing a website for a friend.
